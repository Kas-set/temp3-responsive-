Étape 1: Installation de XAMPP

Téléchargez XAMPP à partir du site officiel (https://www.apachefriends.org/index.html) selon votre système d'exploitation (Windows, macOS, Linux).
Installez XAMPP en suivant les instructions fournies dans le programme d'installation.
Étape 2: Configuration de la base de données

Démarrez XAMPP et assurez-vous que les modules Apache et MySQL sont en cours d'exécution.
Ouvrez votre navigateur et accédez à l'adresse http://localhost/phpmyadmin/ pour accéder à phpMyAdmin.
Créez une nouvelle base de données en spécifiant un nom (par exemple, info_recolt) et choisissez l'encodage approprié (généralement utf8mb4_general_ci).
Dans cette base de données, créez une nouvelle table appelée formulaire avec les colonnes suivantes :
id : int, clé primaire, auto-incrémentée.
nom : varchar.
prenom : varchar.
tel : varchar.
mail : varchar.
sexe : varchar.
Étape 3: Création du formulaire HTML

Créez un fichier HTML (par exemple, formulaire.html) et insérez-y le code suivant :
html
Copy code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Formulaire</title>
</head>
<body>
    <h1>Formulaire d'inscription</h1>
    <form action="traitement.php" method="POST">
        <label for="first_name">Prénom :</label>
        <input type="text" name="first_name" id="first_name" required><br>

        <label for="last_name">Nom :</label>
        <input type="text" name="last_name" id="last_name" required><br>

        <label for="tel">Téléphone :</label>
        <input type="tel" name="tel" id="tel" required><br>

        <label for="mail">Email :</label>
        <input type="email" name="mail" id="mail" required><br>

        <label for="gender">Sexe :</label>
        <select name="gender" id="gender" required>
            <option value="homme">Homme</option>
            <option value="femme">Femme</option>
        </select><br>

        <button type="submit">Envoyer</button>
    </form>
</body>
</html>
